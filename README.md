## 📋 Description

SmartGenChatbot is a modern, free chatbot solution designed to be easy to integrate and use. Built with TypeScript for type safety and JavaScript for flexibility, this project is currently in active development with **ideaband build mode** enabled.

## ✨ Features

- 🤖 Intelligent conversation handling
- ⚡ Fast and responsive
- 🎨 Clean and modern UI
- 📱 Mobile-friendly design
- 🔧 Easy to customize
- 📦 Lightweight and modular
- 🚀 Production-ready

## 🚀 Getting Started

### Prerequisites

- Node.js 14+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/bayeziddev/smartgenchatbot-free.git

# Navigate to the directory
cd smartgenchatbot-free

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm run test
```
